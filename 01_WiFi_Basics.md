# Wi-Fi Basics

## Description

Wi-Fi (Wireless Fidelity) is a wireless networking technology that allows devices to connect to a network and the Internet without using physical cables.

Wi-Fi uses radio waves to transmit and receive data between devices and a wireless router or Access Point (AP).

It is one of the most widely used wireless communication technologies in homes, offices, schools, airports, cafes, and public places.

---

# What is Wi-Fi?

Wi-Fi is a technology based on the IEEE 802.11 standard that enables wireless communication between devices.

Instead of Ethernet cables, Wi-Fi uses radio frequency (RF) signals to exchange data.

Examples of Wi-Fi-enabled devices:

* Laptop
* Smartphone
* Tablet
* Smart TV
* Printer
* IoT Devices
* Gaming Console

---

# Why is Wi-Fi Important?

Wi-Fi provides:

* Wireless Internet Access
* Device Mobility
* Easy Network Expansion
* Cable-Free Communication
* Support for Multiple Devices

---

# History of Wi-Fi

* 1997 – First IEEE 802.11 Standard Released
* 1999 – Wi-Fi Alliance Founded
* 2003 – WPA Introduced
* 2004 – WPA2 Became Standard
* 2018 – WPA3 Announced
* Present – Wi-Fi 6 and Wi-Fi 7 continue improving speed, security, and efficiency.

# How Wi-Fi Works

Wi-Fi allows devices to communicate using radio signals.

Basic process:

```text
Internet
    ↓
Modem
    ↓
Wi-Fi Router / Access Point
    ↓
Wireless Signals
    ↓
Laptop / Phone / Tablet
```

The Access Point sends and receives wireless signals, allowing connected devices to communicate with the network.

---

# Components of a Wi-Fi Network

## 1. Access Point (AP)

The device that broadcasts the wireless network and allows clients to connect.

Example:

```text
Router (AP)
```

---

## 2. Station (STA)

Any wireless client device.

Examples:

* Laptop
* Smartphone
* Tablet

---

## 3. Router

Routes traffic between the local network and the Internet.

---

## 4. Modem

Connects the router to the Internet Service Provider (ISP).

---

## 5. Client Device

Any device using Wi-Fi to access the network.

---

# What is WLAN?

WLAN stands for:

```text
Wireless Local Area Network
```

A WLAN connects devices wirelessly within a limited area.

Examples:

* Home
* School
* Office
* Library

---

# Wi-Fi Uses Radio Waves

Unlike Ethernet, Wi-Fi transmits data using radio frequency signals.

Common frequency bands:

* 2.4 GHz
* 5 GHz
* 6 GHz (Wi-Fi 6E and newer)

These bands are discussed in detail later in this repository.

---

# Wi-Fi Standards Overview

Wi-Fi is based on the IEEE 802.11 family of standards.

Common generations include:

| Wi-Fi Generation | IEEE Standard            |
| ---------------- | ------------------------ |
| Wi-Fi 4          | 802.11n                  |
| Wi-Fi 5          | 802.11ac                 |
| Wi-Fi 6          | 802.11ax                 |
| Wi-Fi 6E         | 802.11ax (6 GHz support) |
| Wi-Fi 7          | 802.11be                 |

# Advantages of Wi-Fi

* No cables required
* Easy installation
* Supports mobility
* Connects multiple devices
* Cost-effective
* Easy to expand

---

# Disadvantages of Wi-Fi

* Signal interference
* Limited range
* Lower speed than wired Ethernet (in many scenarios)
* Security risks if not configured properly
* Signal obstacles (walls, furniture, etc.)

# Wi-Fi vs Ethernet

| Feature      | Wi-Fi                        | Ethernet                          |
| ------------ | ---------------------------- | --------------------------------- |
| Medium       | Wireless                     | Cable                             |
| Mobility     | High                         | Low                               |
| Speed        | Variable                     | Generally Faster                  |
| Installation | Easy                         | Requires Cables                   |
| Security     | Lower (if poorly configured) | Higher (Physical Access Required) |

# Wi-Fi in Cyber Security

Wireless networks are common targets for attackers.

Security professionals study Wi-Fi to:

* Secure wireless networks
* Analyze wireless traffic
* Detect rogue access points
* Investigate attacks
* Perform wireless security assessments (with proper authorization)

Understanding Wi-Fi basics is the foundation before learning advanced wireless security concepts.
